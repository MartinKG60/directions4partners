# Directions for Partners

Two HTML themes for the same landing page.

## Files
- index.html: light theme
- styles.css: light theme styles
- index-alt.html: dark theme
- styles-alt.css: dark theme styles

## Theme switch
Use the floating "Skift tema" button on the left to toggle between the two pages.

## Run locally
Open index.html or index-alt.html in a browser.
